# Contributing

## Local Setup

1. Fork and clone the repository.
2. Create a virtual environment.
3. Install dependencies from `requirements.txt`.
4. Copy `.env.example` to `.env`.
5. Run the app and verify upload flow.

## Pull Requests

- Keep PRs focused and small.
- Include security impact notes for upload/scan changes.
- Update docs when behavior changes.

## Code Style

- Keep code readable and explicit.
- Fail closed on security checks unless explicitly designed otherwise.
- Preserve audit metadata fields and consent logic.
